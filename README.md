# stoozebot-check

## Installation

#### 1. Clone Repository

```sh
git clone https://github.com/PrasitSahu/stoozebot-check.git
cd stoozebot-check
```

#### 2. Install Dependencies

```sh
npm i
```

#### 3. Build And Run

```sh
npm run build
npm start
```

or

```sh
npm run dev
```

> [!NOTE]
> Use the `npm run dev` command only locally, for production use the `npm start`
